# Rozjetí frontendu

je potřeba mít nainstalovaný [Node.js](https://nodejs.org/en/)

## Build aplikace
```bash
npm run build    // vytvoří složku dist se zdrojovými soubory js, css a html
```

## Spuštění

Ve složce ./bdas2-frontend pustit následujicí příkazy:
```bash
npm install     // nainstaluje node_modules 
npm run dev     // rozjede aplikaci a v konzoli vypíše adresu, na které projekt běží
```
